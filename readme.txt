Cordial saludo

Este proyecto no tiene un ejecutable por lo que se debe descargar el código y ejecutar desde el IDe Eclipse Oxygen,por medio de la clase que se encuentra en el paquete runners "ConsultaProductoEbay",
No se encuentra terminado falta capturar los datos en Excel, controlar la excepción en Excel cunado no se encuentra el archivo y el envío del correo.

el objetivo de esta automatizacion es navegar por la página de ebay, realizar una consulta, guardar en un archivo Excel los datos de producto, descripción y precio, seleccionar un producto al azar
y enviarlo hasta el carrito de compras, verificar el producto y llevar la compra hasta antes del pago.
